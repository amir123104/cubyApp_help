Windows says the app is potentially harmful after downloading

Unfortunately, the app is currently not verified by Microsoft, which means that Windows SmartScreen may show a security warning. This is a common occurrence for new applications that haven't been signed with an extended validation certificate.

Solution: Click "More info" in the warning dialog, then select "Run anyway" to proceed with the installation. Rest assured, Cuby is completely safe and does not contain any malware or viruses.

How to install Cuby?

Follow these steps for a successful installation:

  1. Download the installer file (named "downloads-cubyApp.zip")
  2. Important: You don't need to extract the ZIP file
  3. Double-click the ZIP file to open it
  4. Locate and run "setup.exe" inside the ZIP archive
  5. Follow the on-screen installation wizard
  6. Restart your computer if prompted

The installation should begin immediately after running setup.exe. The entire process typically takes 1-2 minutes.

3

Installation fails or shows errors

If you encounter errors during installation, try these troubleshooting steps:

  • Close all applications before installing
  • Run the installer as Administrator (right-click â†’ "Run as administrator")
  • Check that you have at least 2.5 GB of free disk space
  • Ensure you're running Windows 10 or 11 (64-bit)
  • Temporarily disable your antivirus software during installation
  • Make sure .NET Framework 4.8 or later is installed

If problems persist, please contact our support team with a screenshot of the error message.

4

App crashes or doesn't start

If Cuby crashes on launch or doesn't start properly:

  • Update your graphics drivers to the latest version
  • Install all pending Windows Updates
  • Try running the app in compatibility mode
  • Check Windows Event Viewer for detailed error logs
  • Reinstall Cuby completely (uninstall first, then install fresh)
